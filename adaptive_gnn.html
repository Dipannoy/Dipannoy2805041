Trainable Control Networks for Selective Communication

In large-scale crystal structure prediction, the computational cost of full message-passing can be prohibitive. This project involved designing trainable control networks that sit on top of standard GNN architectures to manage selective node communication.

Key Optimization Achievements

20%

Edges Pruned

0%

Accuracy Loss

Methodology

The control network learns which edges in the crystal graph are vital for property prediction and which are redundant. By dynamically pruning these edges during the forward pass, we significantly reduce the number of message-passing operations.

Implementation Highlights

  • Trainable gating mechanisms for adaptive adjacency matrices.
  • Evaluation on high-atom count crystal datasets (JARVIS/Materials Project).
  • Integration with PyTorch Geometric and DGL for efficient sparse matrix operations.

Computational Impact

Reducing the graph density by 20% led to a substantial decrease in training time and memory footprint. This makes the model highly suitable for deployment on high-performance computing (HPC) clusters where resource allocation is critical.
